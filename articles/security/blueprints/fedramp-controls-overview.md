# <a name="fedramp-overview"></a>FedRAMP – přehled

[Federal rizika a autorizace správu Program (FedRAMP)](https://www.fedramp.gov), je program government celou USA, který poskytuje standardizovaná přístup k zabezpečení, autorizace a nepřetržité monitorování produktů cloudu a služby. FedRAMP definuje proces hodnocení zabezpečení podle sada standardních požadavek odvozené od [Národního institutu standardů a technologie (NIST) speciální publikace SP 800 53](https://csrc.nist.gov/publications/detail/sp/800-53/rev-4/final) kontrolních mechanismů pro zabezpečení.
